PART II : CLassList 


-contains check if a tag contain a class or not 

ex : h1.classList.contains("cls") // true or false 


-add add class into a tag 

ex: h1.classList.add("cls")

-remove a class from a tag 

ex: h1.classList.remove("cls")

-Toggle ( add or remove  a class from a tag )

ex: h1.classList.toggle("cls")
